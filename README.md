# lkpd_array
1. Mengapa perlu array?
Untuk menyimpan banyak data dalam satu variabel.


2. Bagaimana array menghemat variabel?
Tanpa array, kita harus buat variabel banyak: nilai1, nilai2, dst. Array cukup satu.


3. Dampak data bertambah?
Ukuran array bertambah, perhitungan rata-rata berubah.


4. Cara menampilkan data tertentu?
Gunakan filter(), indexOf(), atau find().


5. Kesalahan umum?

Salah index

Lupa pakai splice() saat hapus

Mutasi array tanpa sengaja
